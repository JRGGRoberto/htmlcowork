Seguindo o exempo de W3School Fork

Passos

  1. Acesse https://github.com/JRGGRoberto/htmlcowork
  2. Siga a orientação a imagem O fork copiará o projeto para sua base de projetos
  3. Baixe o projeto conforme a imagem, ou por meio do git grafio Há dicas em deste passo em https://www.w3schools.com/git/git_clone.asp?remote=github
  4. Siga os passos do link para configurar se repositório local https://www.w3schools.com/git/git_clone.asp?remote=github
  5. Faça as alterações no arquivo observando o código onde está sendo solicitado as alterações
  6. Faça os commits alterações para o seu Github e depois
  7. Solicite a alteração no repositório inicial com os seguintes pasos
